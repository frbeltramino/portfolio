📦 Instalación y Uso

Clonar el repositorio:

git clone https://github.com/frbeltramino/tu-repo.git
cd tu-repo


Instalar dependencias:

npm install


Ejecutar en desarrollo:

npm run dev


Build de producción:

npm run build
